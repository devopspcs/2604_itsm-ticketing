## How to Use

### 1. Navigate Between Pages
- Click any item in the sidebar to navigate
- Active page is highlighted in blue

### 2. View Issues
- Issues appear as cards in columns
- Each card shows: Issue Key, Title, Status

### 3. Move Issues
- Drag a card to another column
- Card automatically updates status

### 4. View Issue Details
- Click on any card
- Modal opens with full details
- Edit and save changes

### 5. Add New Column
- Scroll to the right
- Click "+ Add Column"
- Enter column name
- Press Enter

### 6. Filter Issues
- Use Quick Filters section
- Search by title or key
- Filter by status, assignee, etc.

## Color Scheme

- **Primary Color**: Blue (active items, buttons)
- **Background**: Light gray/white
- **Text**: Dark gray/black
- **Borders**: Light gray
- **Hover**: Light blue background

## Responsive Design

- **Desktop**: Full layout with sidebar
- **Tablet**: Sidebar collapses to icons
- **Mobile**: Sidebar becomes drawer menu

## Keyboard Shortcuts

- **Enter**: Save changes
- **Escape**: Close modal or cancel
- **Drag**: Move cards between columns
- **Click**: Open details or navigate

## Performance

- **Load Time**: < 2 seconds
- **Drag & Drop**: Smooth 60 FPS
- **Search**: Real-time filtering
- **Responsive**: Instant UI updates
