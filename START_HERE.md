## 🚀 See It In 3 Steps

### Step 1: Start the Application

```bash
# Terminal 1: Backend
cd backend
go run ./cmd/server

# Terminal 2: Frontend (open new terminal)
cd frontend
npm run dev
```

### Step 2: Open Browser

```
http://localhost:3000/projects/{projectId}
```

Replace `{projectId}` with an actual project ID.

### Step 3: See Jira-Style Layout

You'll see:
- ✅ **Left Sidebar** with navigation (Backlog, Board, Sprint, Settings, etc.)
- ✅ **Kanban Board** with columns and cards
- ✅ **Issue Cards** with title, key, and status
- ✅ **Drag & Drop** to move cards between columns
- ✅ **All Jira Features** accessible through navigation
